Apex

Includes search functionality, refresh/resync icons, and detailed views for Apex Classes, Test Classes, and Triggers. Options include viewing test coverage and related test classes.

  • Search Bar: Quickly search through Apex. This search is case insensitive. It will only search though the item for which the filter is selected.
    Which means, if no neither of Apex Test Classes or Apex Trigger checkbox is selected, it will search through only Apex Classes(Non Test Classes). Meanshile if I select Apex Test Classes, it will search through only Apex Test Classes and same for Apex Triggers.
  • Refresh/Resync Icon: If you see the data is not picking up your Apex, try to use this REFRESH icon to resync the data from your salesforce org.
  • Apex Classes: This is the default view when the extension loads or if someone clicks on.
    • Covering Test Classes Toggle
    • Test Coverage Percentage
  • Test Class: Filter and view test classes with detailed results.
    • Covered Class/Trigger Toggle
    • Run Test Class with method status details
  • Triggers: Manage triggers with filtering and coverage info.

Flows

Manage your flows with a search bar, refresh options, and a list that shows active flows including Process Builders, AutoLaunched, and Screen flows. Filters help identify active versus inactive statuses.

  • Search Bar: Locate flows quickly.
  • Refresh/Resync Icon: Synchronize flow data.
  • All Active Flows: Default list with details.
    • Inactive Checkbox Filter
    • Process Builders list (PB)
    • AutoLaunched and Screen flows
    • Icons for Process Type and Active/Inactive status

Debug

Track and analyze your debug traces. Easily search, refresh, and inspect logs with details like start/end dates and tracking status.

  • Search Bar: Find specific debug traces.
  • Refresh/Resync Icon: Update trace logs instantly.
  • All Debug Traces: Detailed view of debug logs.
    • Information Icon for start/end dates
    • Tracking Icon for active trace status
    • Refresh TraceFlag Icon for extending trace duration

Apex Changes

Provides version control for your Apex classes. Easily compare different working copies with version toggles and detailed change logs.

  • Search Bar: Quickly locate working copies.
  • Refresh/Resync Icon: Get the latest changes.
  • Working Copy of Apex Classes:
    • Toggle Version Icon to view various versions
    • Select and compare versions via a dedicated interface

Sites

Displays all Experience Sites in your org. Provides direct links to both the Builder and Workspace for easy management.

  • Search Bar: Quickly search for sites.
  • Refresh/Resync Icon: Update site information.
  • All Experience Sites: List view with direct links to Builder and Workspace.

Branding

Customize your experience with the Branding section. It displays your logo, organisation details, user info, and settings for tailoring the extension’s appearance and behavior.

  • Navforce Logo: Redirects to the personal blog.
  • Organisation Name: Displays instance details on hover.
  • Current User Details: Shows user name and email on hover.
  • Pin/Unpin: Toggle to keep the sidebar visible.
  • Settings: Options to disable code coverage highlight and configure sorting orders.
